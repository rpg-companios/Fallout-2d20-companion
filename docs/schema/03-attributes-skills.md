# Схема: Attributes & Skills

## Атрибуты
Канонические ключи: `STR, END, PER, AGI, INT, CHA, LCK` (S.P.E.C.I.A.L., порядок в коде свой).
База: значение 4, лимиты 4–10 (BASE_MIN/MAX), очков на распределение: 12 (DISTRIBUTION_POINTS).

### Модель значения (уже в Zustand)
`Parameter = { base, modifiers: [{source, value, operation}], total }`.
Это фундамент будущей единой системы модификаторов (modifiers-vision.md).

### Что влияет на атрибут (источники)
| источник | вид | поле/правило |
|---|---|---|
| распределение игрока | base | в пределах лимитов |
| origin/трейт: бонус | `attributeBonus: {STR:2}` | +/- к значению (T-4) |
| origin/трейт: лимиты | `attributeLimits: {STR:{min,max}}` | сдвиг потолка/пола (T-4) |
| доп. очки | `attributePointsBonus: N` | больше очков распределения |
| перки (будущее) | bonus / limit | напр. «Интенсивные тренировки» +1 очко |
| эффекты/химия (будущее) | временный modifier | напр. Buffout +STR на время |

Правило (от владельца): «бонус значения» и «расширение лимита» независимы — нельзя
поднять атрибут выше лимита без отдельного модификатора лимита.

### Лимиты — порядок источников (из #6)
`attributeLimits` (трейт/ориджин) → BASE (4..10). Несколько источников лимитов —
берём самый «широкий»? ❓ AS-1: как складывать несколько модификаторов лимита (max из max? сумма дельт?).

## Навыки
Канонические ключи: ALL_SKILL_KEYS (ATHLETICS, BARTER, BIG_GUNS, ENERGY_WEAPONS,
EXPLOSIVES, LOCKPICK, MEDICINE, MELEE_WEAPONS, PILOT, REPAIR, SCIENCE, SMALL_GUNS,
SNEAK, SPEECH, SURVIVAL, THROWING, UNARMED).
⚠️ Сейчас навыки в коде местами идентифицируются ЛОКАЛИЗОВАННЫМ именем (рус.). Цель:
перейти на canonical-ключи везде (T-3).

База: очки навыков = INT + 9 + (level-1). Тег даёт стартовые 2. Потолок = skillMaxValue (по умолч. 6; на 1 уровне ≤3).

### Что влияет на навык (источники)
| источник | поле |
|---|---|
| распределение игрока | base (в пределах потолка) |
| тег (selected/extra/forced) | стартовые +2 |
| трейт: обязательные теги | `forcedSkills: [...]` |
| трейт: доп. теги | `extraSkills: N` |
| трейт: бонус к навыку | `skillModifiers: {SURVIVAL: 2}` (ключи → canonical, T-3) |
| трейт: потолок | `skillMaxValue: N` |
